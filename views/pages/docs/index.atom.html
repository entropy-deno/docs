getting started

Entropy is a web development backend framework based on and created for applications written on the Deno TypeScript runtime. This chapter will make you familiar with the fundamentals of Entropy, providing you with a quick overview of the most significant features.

It is recommended to read this chapter thoroughly before proceeding.

The following sections of this chapter put an emphasis on introducing to you the basic use cases and patterns for modern backend systems engineering with the use of the Entropy framework.

If you are looking for detailed technical specificaiton of the framework modules or further information on concrete topics, view the navigation bar to the side and feel free to proceed to a point of interest.

prerequisites

Although this chapter aims to convey the basics of Entropy in a beginner-friendly manner, it is nonetheless recommended to consider the following basic prerequisites for understanding presented material before proceeding:

  1. JavaScript & TypeScript - A solid understanding of the main programming language used by Entropy is required to write backend systems with the use of this framework.

  2. Deno - It is recommended to have a solid understanding of the main runtime for backend applications in the Entropy ecosystem. If you are familiar with other runtimes such as node along with npm, Deno is likely to feel natural. You are required to have Deno installed on your device in order to be able to work on Entropy applications.

  3. General Backend Knowledge - It is again recommended to have a solid understanding of common backend patterns and terminology. This is not, however, a prerequisite with heavy emphasis, due to the beginner-friendly introductory nature of this chapter.

    The Entropy learning curve is still (to some extent) dependent on your existing knowledge of backend development beyond explanations of this chapter and selected topics further in this documentation!

the entropy framework

The framework uses TypeScript to provide best type safety. We believe that this language is the best option for secure Deno app development.

Installation

Entropy requires no installation process of any additional tooling for creating new projects. You can use the following command in order to launch Entropy project creator:

      

        

        

        


        
terminal

      
deno run -A
      https://deno.land/x/entropy_cli/init.ts

When the installation is ready, you can launch the project:

      

        

        

        


        
terminal

      
cd my-project  # Your project directory

deno task dev

Now you are able to explore the core concepts of Entropy in next sections.
