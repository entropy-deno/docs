Getting started

In this section you are going to make familiar with Entropy framework and its installation / project creating process.

Introducing Entropy

Entropy is a modern and declarative backend framework for Deno. It makes it easy to create fast and secure web applications with awesome developer experience.

Philosophy and features

Entropy provides a strong architecture which makes it easier to create highly scalable apps.

The framework uses TypeScript to provide best type safety. We believe that this language is the best option for secure Deno app development.

Prerequisites

To learn Entropy you will need to have a basic knowlege of:

  • Deno runtime
  • TypeScript language
  • Basics of HTTP and MVC (optional)

Installation

Entropy requires no installation process for creating new projects. You can use the following command in order to launch Entropy project creator:

      

        

        

        


        
terminal

      
deno run -A https://deno.land/x/entropy_cli/init.ts
